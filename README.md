## Features

- Real-time **keyboard latency** measurement (fastest key press intervals)  
- Real-time **mouse movement polling rate** detection  
- **Fastest mouse polling rate** tracking  
- **Day/Night theme toggle** (Sun/Moon)  
- Lightweight and **stable Tkinter GUI**  
- Ready for compilation into **standalone .exe**  
- Also available compiled .exe for download
